Blog setup guide

  1. Create a blog/ directory in your notebooks folder.
  2. Create a index.qmd file in the root of the blog/ directory. This file signal Quarto that you are creating a blog and config how blogs are indexed. Change your title and subtitle
    Change the listing below in case you creating a different layout in Quarto
## step 1
!mkdir blog

## step 2
!touch blog/index.qmd

content = \
"""---
title: nbdev guide
subtitle: Tips to work with nbdev in your AI project
listing:
    sort: "date desc"
    contents: "posts"
    sort-ui: false
    filter-ui: false
    categories: true
    feed: true
page-layout: full
---"""

with open('./blog/index.qmd','w') as f:
    f.write(content)
